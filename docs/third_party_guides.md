This repo cross-references popular communities and guides, but does **not** endorse everything in them.

Useful for:
- structured Windows configuration checklists
- measurement-first ideas (PresentMon, xperf)
Avoid:
- security regressions (Defender off, blocklist off)
- fragile timer/interrupt hacks without measurement
